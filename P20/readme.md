# const函数
## const成员函数不会修改对象状态


const 函数和非const函数可以构成重载 

## const 成员函数只能访问数据成员的值，不能修改他 当被定义为const 时候是不是就是调用const


# const 对象
 表示该对象的值不能被修改
 const 对象只能调用const成员函数,因为非const 函数有被修改的风险

 # mutable 修饰符
 被mutable修饰的成员变量能被const函数修改，const对象也能修改
