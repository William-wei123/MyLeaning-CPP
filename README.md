#### 构造函数和析构函数
    1. 构造函数：无返回类型，一般为public
    2. 不带参数的构造函数为默认构造函数
    3. 如果定义带参数的构造函数，则系统不会生成默认的构造函数
    4. new运算符，动态申请内存和调用构造函数
